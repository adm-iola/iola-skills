# Маршрутизация по слоям данных

Используй этот skill, когда нужно понять, к какому слою открытых данных
относится вопрос пользователя.

## Правило маршрутизации

Не выбирай слой только по памяти. Если доступен MCP, сначала используй:

```text
layer_suggest
```

Затем:

- для вопроса по конкретной организации используй `layer_answer_context`;
- для статистики и заполненности используй `layer_stats`;
- для списков значений и фильтров используй `layer_facets`;
- для проблем качества используй `quality_summary` или `quality_findings`.

## Текущие слои

- `schools` - школы;
- `kindergartens` - детские сады.

Новые слои должны добавляться без изменения базового правила: сначала выбрать
слой через MCP, затем получить контекст ответа через MCP.

## Ответ пользователю

Обычно не рассказывай пользователю о внутренней маршрутизации. Просто дай ответ
по найденным данным. Объясняй выбор слоя только если пользователь спрашивает,
почему выбран именно этот источник.
